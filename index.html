🚀 FormClue Chrome Extension

Autofill and AI suggestions for Google Forms.

Latest version: v1.0

📽️ How to Use

🛠️ Installation Steps

  1. Download and extract the ZIP file above.
  2. Open chrome://extensions/ in Chrome.
  3. Enable Developer Mode (top-right).
  4. Click "Load Unpacked" and select the folder you extracted.
  5. You're done! Open a Google Form to start using FormClue.

✨ Features

  • 🔒 Secure autofill of personal data
  • 🧠 AI-suggested answers based on question type
  • 🚀 Works instantly on any Google Form
  • 🧩 Easy one-click installation
